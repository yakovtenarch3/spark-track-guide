# תכונת שני טורים (Two Column Mode)

## סקירה כללית
תכונה חדשה המאפשרת קריאה נוחה יותר של מסמכים באמצעות פיצול התוכן לשני טורים, בדומה למצב קריאה עיתון.

## רכיבים נתמכים

### ✅ RichDocumentEditor (Word/HTML Documents)
עורך המסמכים תומך כעת במצב קריאה בשני טורים:

#### תכונות:
- **כפתור Toggle**: כפתור מיוחד בסרגל הכלים עם אייקון 📰/📄
- **מעבר חלק**: מעבר חלק בין מצב טור יחיד (1) לשני טורים (2)
- **CSS Columns**: שימוש ב-CSS columns לפיצול אוטומטי של התוכן
- **קו מפריד**: קו אנכי בין הטורים לבהירות
- **יישור מוצדק**: טקסט מיושר לשני הצדדים במצב שני טורים

#### איך להפעיל:
1. פתח מסמך Word/HTML בעורך
2. לחץ על כפתור "2 טורים" / "1 טור" בסרגל הכלים העליון
3. התוכן יתחלק אוטומטית לשני טורים

#### קוד רלוונטי:
```typescript
// State
const [twoColumnMode, setTwoColumnMode] = useState(false);

// Toggle Button
<Button
  variant="ghost"
  size="sm"
  className="gap-1 text-xs h-8"
  onClick={() => setTwoColumnMode(!twoColumnMode)}
  title={twoColumnMode ? "טור יחיד" : "שני טורים"}
>
  {twoColumnMode ? "📄" : "📰"} {twoColumnMode ? "1" : "2"} טורים
</Button>

// CSS Application
<Card 
  className={`... ${
    twoColumnMode 
      ? 'columns-2 gap-8 column-fill-auto' 
      : ''
  }`}
  style={twoColumnMode ? {
    columnRule: '1px solid #e5e7eb'
  } : {}}
>
```

### ⏳ LuxuryPDFReader (PDF Documents)
הוכן לתמיכה עתידית. עבור PDF, "שני טורים" ימשמע הצגת שני עמודים זה לצד זה (2-up view).

## יתרונות

### 📖 קריאה טובה יותר
- רוחב שורה קצר יותר מקל על מעקב אחר הטקסט
- פחות תזוזות עיניים אופקיות
- קריאה מהירה יותר

### 🖥️ ניצול מסך
- שימוש יעיל יותר במסכים רחבים
- תצוגה של יותר תוכן בו-זמנית
- פחות גלילה נדרשת

### 📰 חוויית קריאה מוכרת
- דומה לקריאת עיתון או מגזין
- נוח לקריאה ממושכת
- עיצוב מקצועי ומעוצב

## טכנולוגיה

### CSS Columns
```css
.two-column-mode {
  columns: 2;
  gap: 2rem; /* 32px */
  column-fill: auto;
  column-rule: 1px solid #e5e7eb;
  text-align: justify;
  line-height: 1.6;
}
```

### Tailwind Classes
- `columns-2`: פיצול לשני טורים
- `gap-8`: מרווח בין טורים (32px)
- `column-fill-auto`: מילוי טורים באופן אוטומטי

## מגבלות ושיקולים

### ⚠️ מגבלות ידועות
1. **רוחב מסך**: עובד טוב רק במסכים רחבים (>1024px)
2. **כותרות ארוכות**: כותרות ארוכות עשויות להיראות לא אופטימלית
3. **תמונות**: תמונות גדולות עשויות לשבור את הפריסה
4. **טבלאות**: טבלאות רחבות לא יפוצלו בצורה אופטימלית

### 💡 המלצות שימוש
- השתמש במצב זה במסכים רחבים (Desktop)
- מתאים במיוחד למסמכי טקסט רציפים
- פחות מתאים למסמכים עם הרבה תמונות או טבלאות

## פיתוחים עתידיים

### 🚀 תכונות מתוכננות
1. **PDF 2-Up View**: הצגת שני עמודי PDF זה לצד זה
2. **מצב 3 טורים**: אופציה נוספת ל-3 טורים
3. **רוחב טור מותאם אישית**: שליטה ידנית על רוחב הטורים
4. **שמירת העדפות**: שמירת בחירת המשתמש ב-localStorage
5. **Responsive**: התאמה אוטומטית למסכים קטנים יותר
