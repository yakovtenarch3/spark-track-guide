# 🎨 מערכת ערכות נושא מתקדמת

## תכונות

### יצוא ויבוא ערכות נושא
- **יצוא**: כפתור Download ליד כל ערכת נושא ליצירת קובץ JSON
- **יבוא**: כפתור Upload להעלאת ערכות נושא מקבצים

### עריכת ערכות נושא
כל ערכת נושא מותאמת אישית מקבלת כפתור עריכה 🖊️

### אפשרויות התאמה אישית

#### טאב 1: צבעים בסיסיים
- צבע רקע
- צבע ראשי
- צבע משני
- צבע כרטיסים
- צבע גבול כרטיסים
- צבע טקסט
- צבע כותרות

#### טאב 2: מתקדם
- צבע הדגשה
- רקע כפתורים
- טקסט כפתורים
- **סיידבר**:
  - רקע סיידבר
  - טקסט סיידבר
  - גבול סיידבר

#### טאב 3: עיצוב (חדש!)
- **עובי מסגרת**: 0-10px (Slider)
- **רדיוס פינות**: 0-50px (Slider)
- **עוצמת צל**: 0-100% (Slider)
- **משקל גופן**: 300-700 (Select)

## שימוש

### יצוא ערכת נושא
1. פתח תפריט ערכות נושא
2. לחץ על כפתור ה-Download ליד הערכה הרצויה
3. הקובץ JSON יישמר במחשב

### יבוא ערכת נושא
1. פתח תפריט ערכות נושא
2. לחץ "יבוא ערכת נושא"
3. בחר קובץ JSON
4. הערכה תתווסף אוטומטית

### עריכת ערכת נושא
1. פתח תפריט ערכות נושא
2. לחץ על כפתור העריכה (🖊️) ליד ערכה מותאמת
3. ערוך צבעים ועיצוב
4. שמור שינויים

# 🛠️ מצב פיתוח (Developer Mode)

## תכונות

### הפעלה והשבתה
- Toggle ראשי בהגדרות
- שמירת מצב ב-localStorage
- כיבוי מצב פיתוח משבית אוטומטית את כל הכלים

### כלים זמינים

#### 1. קונסול (Console)
- הצגת לוגים במסך
- מעקב אחר פעולות
- שגיאות וניפוי באגים
- **איך להפעיל**: הגדרות → מצב פיתוח → קונסול

#### 2. אלמנטור (Element Inspector)
- בדיקת מבנה DOM
- מידע על אלמנטים
- צילומי מסך
- קואורדינטות ומיקום
- **איך להפעיל**: הגדרות → מצב פיתוח → אלמנטור

## איך להשתמש

### הפעלת מצב פיתוח
1. עבור ל**הגדרות**
2. חפש את כרטיס "מצב פיתוח" (כתום)
3. הפעל את ה-Toggle הראשי
4. בחר כלים נוספים לפי הצורך

### שימוש בקונסול
```typescript
// המערכת תתעד אוטומטית:
- כניסות למערכת
- פעולות על משימות
- שגיאות ואזהרות
- פעולות CRUD
```

### שימוש באלמנטור
1. הפעל את האלמנטור
2. לחץ על כפתור ה-Bug
3. העבר עכבר מעל אלמנטים
4. לחץ לבחירה ולקבלת פרטים
5. צלם מסך של האלמנט
6. העתק פרטים ללוח

## אזהרות

⚠️ **השפעה על ביצועים**
כלי הפיתוח עשויים להאט את האפליקציה.
השתמש רק לצורך ניפוי באגים.

⚠️ **אבטחה**
אל תשאיר מצב פיתוח פעיל בייצור (Production).

## API

### useDeveloperMode Hook

```typescript
import { useDeveloperMode } from '@/hooks/useDeveloperMode';

function MyComponent() {
  const { 
    enabled,           // האם מצב פיתוח פעיל
    consoleEnabled,    // האם הקונסול פעיל
    inspectorEnabled,  // האם האלמנטור פעיל
    toggleDevMode,     // הפעל/כבה מצב פיתוח
    toggleConsole,     // הפעל/כבה קונסול
    toggleInspector    // הפעל/כבה אלמנטור
  } = useDeveloperMode();

  // שימוש...
  if (enabled) {
    console.log('מצב פיתוח פעיל');
  }
}
```

## אינטגרציה

### DevInspector
```typescript
// הרכיב מגיב אוטומטית למצב הפיתוח
<DevInspector />

// לא יוצג אם מצב פיתוח מכובה
// מעודכן אוטומטית כשמשנים הגדרות
```

## טיפים

### 1. ניפוי באגים מהיר
```
הגדרות → מצב פיתוח → קונסול + אלמנטור
```

### 2. בדיקת עיצוב
```
אלמנטור → לחץ על אלמנט → ראה CSS + מיקום
```

### 3. מעקב אחר שגיאות
```
קונסול → עוקב אוטומטית אחר כל הלוגים
```

### 4. שיתוף באגים
```
אלמנטור → צלם מסך → שלח לצוות
```

## דוגמאות שימוש

### בדיקת רכיב מסוים
1. הפעל אלמנטור
2. נווט לעמוד עם הרכיב
3. לחץ על הרכיב
4. בדוק Props, State, CSS

### מעקב אחר API Call
1. הפעל קונסול
2. בצע פעולה (למשל: שמור משימה)
3. ראה בקונסול את ה-request וה-response

### דוח באג מפורט
1. שחזר את הבאג
2. צלם מסך עם האלמנטור
3. העתק לוגים מהקונסול
4. שלח דוח עם כל המידע
